Google Maps JavaScript API permite personalizar mapas con tu propio contenido y mostrarlo en páginas web y dispositivos móviles. Permite 4 tipos básicos de mapas (ruta, satélite, híbrido y terreno), los cuales pueden modificarse utilizando capas y estilos, controles y eventos, servicios y librerías.

API KEY


Para poder utilizar la api (aplicaciones móviles, sitios web) es necesaria una API KEY proporcionada por Google

Ejemplo

  • Utilizar la declaración de HTML 5
  • Crear div con el id "map"
  • Definir una funcion para inicializar el mapa
  • Insertar el tag para cargar el JavaScript del API

1.- Center : coordenadas iniciales

2.- Zoom

SVG

Gráficos vectoriales escalables, o Gráficos vectoriales redimensionables o SVG es un formato de gráficos vectoriales bidimensionales, tanto estáticos como animados, en formato XML

La imagen vectorial puede ser redimensionada, tanto como se requiera, sin pérdida de calidad de imagen. Esto no es así con un mapa de bits.

  • Elementos geométricos
  • Mapas de Bits
  • Texto
